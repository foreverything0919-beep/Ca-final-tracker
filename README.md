# CA Final War Room 🎯

**Live Exam Prep Tracker for CA Final (October 15, 2026)**

## Features

✅ **17-Week Study Plan**
- Daily 9-slot tracking (4 morning + 5 afternoon)
- Subject rotation schedule: Audit, FR, IDT, DT, AFM
- Real-time progress visualization

✅ **4 Views**
- **Dashboard**: Overall progress, subject breakdown, 17-week journey
- **Tracker**: Week-by-week slot completion with daily & bar charts
- **Why Missed**: Reason breakdown (personal, social, health, family)
- **Mock Tests**: Score tracking with pass/fail indicators

✅ **Smart Analytics**
- Subject-wise performance tracking
- Missed session pattern analysis
- Urgency-based color coding (green → yellow → red)
- Sunday full revision checkpoints

✅ **Offline-First**
- Progressive Web App (PWA)
- Local storage persistence
- Works fully offline

## How to Use

1. **Dashboard**: View overall progress and week status
2. **Tracker**: Mark sessions as ☐ (empty) → ✓ (done) → ✗ (missed with reason)
3. **Why Missed**: Track patterns (personal outing, social, health, family)
4. **Mock Tests**: Log your practice test scores (5 per subject)

## Data Storage

All data is saved locally in your browser's localStorage. No server uploads—your progress is private.
